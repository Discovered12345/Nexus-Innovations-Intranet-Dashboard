# Nexus Innovations Intranet Dashboard

A modern, responsive intranet dashboard designed for tech teams and developers. This project combines clean UI design with practical functionality for workplace collaboration.

---

## Features

- **Personalized Dashboard**: Welcome message and user-specific widgets
- **Interactive Calendar**: Upcoming events and scheduling
- **Team Collaboration**: Team member status and quick communication
- **Resource Hub**: Organized documentation and tools
- **Developer Focus**: GitHub integration and tech news feed
- **Accessibility**: Dark/light mode and text scaling options

---

## Technologies Used

- HTML5 (Semantic structure)
- CSS3 (Flexbox, Grid, Variables)
- JavaScript (DOM manipulation, event handling)
- Font Awesome (Icons)
